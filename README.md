## 👩🏻‍💻👨🏻‍💻 Development

Please sure that you have below environments

- Node JS >= 10.15.3
- NPM >= 6.13.0
- Yarn >= 1.16.0 (Recommended)

## ⚙️ Setup

Install node packages;

`yarn install`

Then run

`yarn dev`

for development
